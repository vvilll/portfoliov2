About Me

Currently I am studying Computer Science at NIC/UofI with a passion for cyber security and coding. I am passionate about anything related to problem solving and troubleshooting. I enjoy learning new skills and technologies and continually working towards improving.


A few languages I have experience with are

  • C/C++
  • Python
  • HTML/CSS
  • Bash

Additionally I have experience with tools such as

  • Wireshark
  • VirtualBox
  • Git

Interests

Cyber Security is a sector of computer science that I focus on. It is a key part of building software and an area that I enjoying learning about.

Through some of my projects I have also done some game development which is another area of CS I enjoy. The security systems put in place to protect games from malicious users is an area of development I want to learn more about.
